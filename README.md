# simple_chatbot_using_openai
# 🤖 Simple OpenAI Chatbot

A lightweight conversational AI chatbot built with Python and the OpenAI API. This project demonstrates how to integrate OpenAI language models into a Python application using prompt engineering, environment-based API configuration, and a reusable chatbot function.

---

## 📖 Overview

This project provides a simple implementation of an AI chatbot powered by OpenAI's GPT models. It serves as a foundation for learning how conversational AI applications are built while following secure API key management practices.

The chatbot:

- Accepts user questions
- Sends requests to the OpenAI API
- Uses a configurable system prompt
- Returns natural language responses
- Stores API credentials securely using environment variables

---

## ✨ Features

- OpenAI GPT integration
- Python implementation
- Environment variable configuration
- Prompt engineering using system messages
- Reusable chatbot function
- Easy to extend for larger AI applications
